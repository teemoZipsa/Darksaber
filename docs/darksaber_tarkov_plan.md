# ⚔️ Darksaber : Extraction - 기획 및 구현 방향성 정리 (Implementation Plan)

> **💡 [인공지능(AI) 작업을 위한 컨텍스트 릴레이 지침]**
> 다음 채팅 세션을 이어받은 AI 에이전트는 사용자가 이 문서(`implementation_plan.md`)와 함께 제시할 **'닥세월드(라피스) 대륙 원작 지도' 이미지 파일**을 최우선으로 분석하십시오.
> 이 이미지는 단순 참고용 텍스처가 아니라, 현재 절차적 무한 맵 생성 코드를 대체할 **'이미지 기반 베이스 맵 마스크(프레임)'**의 픽셀 데이터입니다. 
> `WorldMap.ts`의 맵 생성 로직을 이 이미지 기반(가장자리 바다 처리 등)의 거대한 고정 사이즈 대륙으로 리팩토링하는 작업(아래 Action Item 1번)부터 즉시 주도적으로 착수하십시오.

## 🗺️ 1. 맵 시스템 개편 (원작 지도 기반 지형 렌더링)
**[현상]** 현재 `src/map/WorldMap.ts`는 노이즈를 바탕으로 무한히 대륙을 찍어내는 절차적 오픈월드 방식이라, 타르코프식 유저 간 동선 조우율이 떨어지며 원작의 향수가 반감됨.

**[수정 방향: 이미지 기반 바이옴(Biome) 맵 씌우기]**
- **원작 맵 완벽 복원:** 제공받은 닥세월드(라피스) 오리지널 대륙의 전경 맵 이미지를 아주 작은 픽셀 크기(예: 100x100 픽셀)로 축소 및 데이터화하여, 맵 생성의 기초 도면(베이스 마스크)으로 사용. (1픽셀 = 1 Chunk 배열)
- **로직 변경:** `WorldMap.ts`청크 생성 함수부에서 현재 렌더링할 청크 좌표의 베이스 맵 픽셀 색상 값을 판별하도록 읽기 함수 덧씌우기.
- **지형 속성 부여:** 픽셀이 🟦파란색이나 바다 대역이면 청크 전체를 바다/벽 타일로 덮어 한계선(Boundary)을 굳히고, 육지/숲 대역이면 내부 노이즈를 굴려 디테일을 무작위 생성함.
- **결과:** 전체 대륙의 굴곡진 베이스 프레임은 닥세월드 원작과 100% 흡사하지만, 안에서 밟는 세부 타일의 배치는 매번 절차적으로 다르게 짜이는 압도적인 배경 완성.

## 🏃 2. 탈출구 (Extraction) 및 은신처 (Safe Zone) 룰
**[수정 방향: 허브 & 레이드 구조 (MMORPG 감성 유지)]**
- **마을/성곽이 곧 은신처(Hub):** 살아서 탈출구인 '마을/성곽' 트거 타일 내부에 진입하면, 유저들끼리 모여 창고를 정리하고 퀘스트를 받는 평화로운 안전지대 커뮤니티(로비) 공간으로 전환.
- **가혹한 출격 (파밍 시작):** 마을 밖 필드(레이드 구역)로 나서는 순간, 소지한 모든 장비와 아이템은 사망 시 100% 드랍하는 생존 룰 적용.
- **얌체 꼼수 방지:** 출격했던 A마을로는 귀환 불가. 목숨을 걸고 거친 필드(10~15분 거리)를 가로질러 다른 B마을 입구 타일에 도달해야만 살아서 귀환.

## 📜 3. 퀘스트(Task) 시스템 편입 전략
* **[메인 시나리오 (계정 영구 성장):]** 특정 구역(던전 최하층 등) 공주/중요 인물 구출 탈출 ➡️ 은신처에 '치료소/전직 상점' 등 영구 해금.
* **[반복 일일 파밍:]** 상인이 의뢰한 잡동사니 전리품을 뜯어와서 장비 수리/포션 자금(골드) 수급.

## 🗝️ 4. 고가치 파밍 및 특수 기믹 (텐션 극대화)
- **[봉인된 방과 마스터키 (마크드 룸)]**
  - 아주 낮은 확률로 얻는 1회용 열쇠로 필드의 특정 방(타일)을 개방해 최상급 유물 100% 확정 상자 파밍.
- **[저주받은 유물 (Cursed Artifact)]**
  - 비싸게 팔리지만 줍는 순간부터 빠져나올 때까지 매 턴 체력 감소 및 행동력 회복 대폭 저하(디버프)를 감수해야 하는 아이템 시스템.

## ⚙️ 5. 필드 생존 시스템 (턴제 퍼즐)
- **환경 함정 (독 늪지대 등):** 사냥터 무단횡단을 방해하기 위해 구석에 독 늪지대가 깔림. 이 타일을 밟으면 턴 페널티 증가 (맵 생성 시 낮은 확률 노이즈로 흩뿌림 체계 적용).
- **무게 페널티:** 파밍 아이템으로 인벤토리가 무거워질수록 행동력(ATB) 게이지 회복 속도 최하락.
